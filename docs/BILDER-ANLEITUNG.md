Bevor Sie beginnen, stellen Sie sicher, dass:

- [ ] Sie Microsoft Edge oder Google Chrome verwenden (empfohlene Browser)

## Bildanforderungen und Best Practices

### Technische Spezifikationen

| Element | Format | Empfohlene Grosse | Max. Dateigrosse |
|---------|--------|-------------------|------------------|
| **Hero-Banner** | JPG | 1920x600 px | 500 KB |
| **Site-Logo** | PNG (transparent) | 400x80 px oder 96x96 px | 100 KB |
| **Countdown-Timer** | JPG | 1200x800 px | 400 KB |
| **Team-Bilder** | JPG | 1200x800 px | 400 KB |
| **Mitarbeiter-Portraits** | JPG | 800x800 px (quadratisch) | 200 KB |
| **Vorher-Nachher** | JPG | 1600x900 px | 500 KB |
| **"Bekannt aus" Logos** | PNG (transparent) | 200x100 px | 50 KB |

### Bildoptimierung vor dem Upload

- Komprimieren Sie grosse Bilder mit Tools wie **TinyPNG** oder **Squoosh**
- Achten Sie auf **hohe Auflosung**, aber moderate Dateigrosse
- Verwenden Sie **aussagekraftige Dateinamen** (z.B. `hero_startseite_2025.jpg`)
- Vermeiden Sie Sonderzeichen und Umlaute in Dateinamen
- Verwenden Sie **PNG-Dateien** fur Logos mit transparentem Hintergrund
- Verwenden Sie **JPG-Dateien** fur Fotos und Hintergrundbilder

## Schritt 1: Bilder in SharePoint hochladen

### 1.1 Zur Dokumentenbibliothek navigieren

1. Offnen Sie: https://fassadenfix2401.sharepoint.com/sites/FassadenFixWissens-OS/SiteAssets
2. Klicken Sie auf **"Hochladen"** -> **"Dateien"**
3. Wahlen Sie die Bilder aus dem Ordner **"Intranet_Assets"** in Google Drive

### 1.2 Empfohlene Bilder fur die Startseite

| Verwendung | Empfohlenes Bild | Pfad in Google Drive |
|------------|------------------|----------------------|
| **Hero-Banner** | Header_Startseite_Fassadenfix.jpg | FassadenFix/Marketing/Intranet_Assets/headers/ |
| **Countdown-Timer** | team_fassadenfix.jpg | FassadenFix/Marketing/Intranet_Assets/team/ |
| **Team-Bereich** | FassadenFixTeamfoto.jpg | FassadenFix/Marketing/Intranet_Assets/team/ |

### 1.3 Alternative: Bilder per OneDrive Sync hochladen

Fur grossere Mengen an Bildern:

1. **OneDrive-Synchronisation aktivieren**
   - Offnen Sie die SharePoint-Dokumentenbibliothek
   - Klicken Sie auf **"Synchronisieren"**
   - Der Ordner wird mit Ihrem lokalen OneDrive-Ordner synchronisiert

2. **Bilder per Drag & Drop kopieren**
   - Offnen Sie den synchronisierten Ordner im Explorer
   - Kopieren Sie alle Bilder aus Google Drive hinein
   - Die Synchronisation erfolgt automatisch

**Vorteil:** Schneller Upload mehrerer Dateien gleichzeitig

## Schritt 2: Hero-Banner-Bild andern

1. Klicken Sie auf **"Bearbeiten"** in der oberen Menuleiste
2. Klicken Sie auf den **Hero-Bereich** (grosses Banner oben)
3. Klicken Sie auf **"Eigenschaften bearbeiten"** (Stift-Symbol)
4. Klicken Sie auf den Layer **"Willkommen bei FassadenFix"**
5. Scrollen Sie nach unten zu **"Hintergrundbild"**
6. Klicken Sie auf **"Andern"**
7. Wahlen Sie **"Hochladen"** und laden Sie **Header_Startseite_Fassadenfix.jpg** hoch
8. Wahlen Sie unter **"Bildposition"** die Option **"Mittig"** oder **"Anpassen"**
9. Klicken Sie auf **"Speichern und schliessen"**
10. Klicken Sie auf **"Seite veroffentlichen"** (wichtig!)

**Tipp:** Verwenden Sie Bilder mit zentrierten Motiven, die auf Mobilgeraten nicht abgeschnitten werden.

---

## Schritt 3: Site-Logo andern

1. Klicken Sie auf das **Zahnrad-Symbol** (Einstellungen) oben rechts
2. Wahlen Sie **"Websiteinformationen"**
3. Klicken Sie auf **"Logo andern"**
4. Laden Sie **FassadenFix_Logo_bunt_transparent.png** hoch
5. Passen Sie bei Bedarf die Zuschneidung an
6. Klicken Sie auf **"Speichern"**

**Hinweis:** Das Logo wird automatisch auf die richtige Grosse skaliert.

---

## Schritt 4: Countdown-Timer-Bild andern

1. Klicken Sie auf **"Bearbeiten"**
2. Klicken Sie auf den **Countdown-Timer-Bereich**
3. Klicken Sie auf **"Details bearbeiten"** (Stift-Symbol)
4. Klicken Sie auf **"Bild andern"**
5. Laden Sie **team_fassadenfix.jpg** hoch
6. Klicken Sie auf **"Speichern und schliessen"**
7. Klicken Sie auf **"Seite veroffentlichen"**

## Anderungen ruckgangig machen

SharePoint speichert automatisch Versionen Ihrer Seiten.

### So stellen Sie eine fruhere Version wieder her:

1. Klicken Sie auf **"Seiteninformationen"** (i-Symbol oben rechts)
2. Wahlen Sie **"Versionsverlauf"**
3. Klicken Sie auf die gewunschte Version, um eine Vorschau anzuzeigen
4. Wahlen Sie **"Wiederherstellen"**, um diese Version wiederherzustellen
5. Bestatigen Sie mit **"Ja"**

**Tipp:** Erstellen Sie vor grosseren Anderungen immer eine Notiz im Versionsverlauf (z.B. "vor Bildwechsel Hero-Banner").

## Haufige Probleme und Losungen

### Problem: Bild wird nicht hochgeladen

**Losung:**
- Uberprufen Sie die Dateigrosse (max. 10 MB pro Datei in SharePoint)
- Stellen Sie sicher, dass der Dateiname keine Sonderzeichen enthalt
- Versuchen Sie es mit einem anderen Browser (Edge oder Chrome empfohlen)
- Prufen Sie Ihre Internetverbindung
- Versuchen Sie den Upload zu einem spateren Zeitpunkt (Server konnte ausgelastet sein)

### Problem: Bild wird pixelig oder verzerrt angezeigt

**Losung:**
- Laden Sie eine hohere Auflosung hoch (mind. 1920px Breite fur Hero-Banner)
- Prufen Sie das Seitenverhaltnis (empfohlen: 16:9 fur Banner, 1:1 fur Portraits)
- Verwenden Sie das JPG-Format mit hoher Qualitat (90% Kompression)
- Vermeiden Sie mehrfaches Skalieren des Bildes

### Problem: Hero-Banner zeigt nur einen Ausschnitt

**Losung:**
- Wahlen Sie unter **"Bildposition"** die Option **"Mittig"** oder **"Anpassen"**
- Verwenden Sie die empfohlene Grosse von 1920x600 Pixel
- Achten Sie darauf, dass wichtige Bildelemente zentral platziert sind
- Testen Sie verschiedene Bildpositionen: "Oben", "Mittig", "Unten"

### Problem: Anderungen werden nicht gespeichert

**Losung:**
- Klicken Sie auf **"Seite veroffentlichen"** (nicht nur "Speichern")
- Leeren Sie den Browser-Cache (Strg + Shift + R)
- Melden Sie sich ab und wieder an
- Prufen Sie, ob Sie Editor-Rechte haben
- Versuchen Sie es im privaten/Inkognito-Modus des Browsers

### Problem: Keine Berechtigung zum Hochladen

**Losung:**
- Wenden Sie sich an den SharePoint-Administrator (Alexander Retzlaff)
- Mindestens **"Mitwirkender"**-Berechtigung erforderlich
- Prufen Sie, ob Sie in der richtigen SharePoint-Gruppe sind
- Versuchen Sie, sich erneut anzumelden

### Problem: Logo wird nicht transparent angezeigt

**Losung:**
- Stellen Sie sicher, dass Sie eine PNG-Datei mit Transparenz verwenden
- Offnen Sie die Datei in einem Bildbearbeitungsprogramm und prufen Sie die Transparenz
- SharePoint unterstutzt nur PNG-Transparenz (nicht GIF)
- Verwenden Sie die bereitgestellten Logo-Dateien aus dem `/logos/` Ordner

### Problem: Bilder laden langsam

**Losung:**
- Komprimieren Sie die Bilder vor dem Upload
- Verwenden Sie die empfohlenen Dateigrossen (siehe Tabelle oben)
- Prufen Sie Ihre Internetgeschwindigkeit
- Vermeiden Sie unnotig grosse Bilddateien (uber 1 MB)

### Problem: Countdown-Timer zeigt falsches Datum

**Losung:**
- Dies ist ein separates Problem (nicht bildbezogen)
- Bearbeiten Sie die Countdown-Timer-Einstellungen
- Prufen Sie das Zieldatum und die Zeitzone
- Kontaktieren Sie den IT-Support

## Best Practices fur Bildpflege

### Regelmaessige Wartung

- **Monatlich:** Uberprufen Sie alle Bilder auf Aktualitat
- **Quartalsweise:** Aktualisieren Sie Team-Fotos bei Anderungen
- **Jahrlich:** Erneuern Sie Hero-Banner und Startseiten-Bilder

### Namenskonventionen

Verwenden Sie konsistente Dateinamen:
- **Hero-Banner:** `hero_[bereich]_[jahr].jpg` (z.B. `hero_startseite_2025.jpg`)
- **Mitarbeiter:** `[vorname]_[nachname].jpg` (z.B. `daniel_janowicz.jpg`)
- **Team:** `team_[abteilung]_[jahr].jpg` (z.B. `team_administration_2025.jpg`)

### Qualitatskontrolle

Vor dem Upload prufen:
- [ ] Bild ist scharf und nicht verwackelt
- [ ] Farben entsprechen dem Corporate Design
- [ ] Keine storenden Elemente im Hintergrund
- [ ] Text (falls vorhanden) ist gut lesbar
- [ ] Dateigrosse ist optimiert
- [ ] Dateiname folgt den Konventionen

### Archivierung

- Legen Sie alte Bilder in einem Unterordner **"Archiv"** ab
- Versehen Sie archivierte Bilder mit Datum (z.B. `hero_startseite_2024_archiv.jpg`)
- Behalten Sie mindestens die letzten 3 Versionen jedes Bildes
